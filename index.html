What is Devstagram?

Devstagram is a programming project sharing platform, but it focuses purely on the "result" of the code and the visuals. You can scroll trough other projects and upload your owns and get noticed for your work. On Devstagram you can start getting credit for those sleepless night coding sessions. In the first version you will be able to upload html, css and javascript projects, but python and C# will be added too.
